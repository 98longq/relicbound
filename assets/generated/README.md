# Generated Art Assets

This folder stores the source art sheets generated for Relicbound.

## Current files

- `hero_sheet.png`
- `enemy_sheet.png`
- `item_sheet.png`
- `environment_sheet.png`

## Purpose

These files are source sheets. They are not final runtime sprites yet.

Planned usage:

- `hero_sheet.png`: player animation reference and future cropped player sprites
- `enemy_sheet.png`: common enemy and boss visual reference
- `item_sheet.png`: pickup, weapon, equipment, and inventory icon source
- `environment_sheet.png`: platform, ground, prop, and background source
