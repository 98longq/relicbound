# 游戏设计文档：Relicbound / 遗物之契

最后更新：2026-06-25
当前阶段：M0 项目规划与基础框架

## 1. 游戏定位

《Relicbound / 遗物之契》是一款 2D 横板动作刷宝 Roguelite 游戏。

玩家在横板关卡中移动、跳跃、攻击、闪避，击败怪物后获得金币、装备、材料和遗物，通过随机掉落形成不同 Build，最终挑战 Boss。

## 2. 核心体验

本项目的核心体验是：

```text
打怪爽 → 掉宝爽 → 变强爽 → Build 成型爽 → 挑战更强敌人
```

开发时优先保证以下感受：

1. 移动和攻击手感清楚。
2. 怪物死亡和掉落反馈明显。
3. 装备与遗物能带来肉眼可见的玩法变化。
4. 每一局都有不同成长路线。

## 3. 世界观

远古时代，人类曾依靠“遗物”获得近似神明的力量。后来遗物失控，深渊裂隙打开，怪物从地下、梦境与虚空中涌出。

玩家是一名被遗物选中的契约者。契约者可以吸收遗物力量，但也会逐渐被侵蚀。玩家进入遗迹，一方面为了寻找宝物和力量，另一方面为了寻找解除契约的方法。

## 4. 主角设定

### 初始角色：无名契约者

- 武器：单手剑 / 短刃
- 定位：平衡型近战角色
- 初始能力：移动、跳跃、普通攻击、闪避、交互
- 成长方式：装备属性 + 遗物效果 + 技能符文

MVP 阶段只实现一个角色。后续可扩展剑士、枪手、法师、狂战士、机械师、影刺客等角色。

## 5. 基础操作

| 输入 | 功能 |
|---|---|
| A / 左方向键 | 向左移动 |
| D / 右方向键 | 向右移动 |
| Space | 跳跃 |
| J / 鼠标左键 | 普通攻击 |
| K | 主动技能 |
| Shift | 闪避 |
| E | 拾取 / 交互 |
| I | 背包 |
| Esc | 暂停 |

输入映射以 Godot InputMap 为准，实际按键可在后续设置菜单中修改。

## 6. 战斗设计

### 6.1 普通攻击

MVP 阶段先实现单段攻击。后续扩展为三段连击：

1. 第一击：快速横斩。
2. 第二击：上挑或斜斩。
3. 第三击：重斩，造成击退。

### 6.2 闪避

闪避是核心生存动作。

- 有短距离位移。
- 有冷却时间。
- 后续可加入短暂无敌帧。
- 闪避方向默认跟随当前移动方向或角色朝向。

### 6.3 技能

MVP 可以暂时不做复杂技能系统，只预留结构。

第一批技能方向：

- 突进斩：向前突进并造成伤害。
- 圆形斩击：对周围敌人造成范围伤害。
- 遗物爆发：消耗能量造成高伤害。

## 7. 怪物设计

MVP 目标先实现 3 种普通怪、1 种精英怪和 1 个 Boss。

| 怪物 | 行为 | 阶段 |
|---|---|---|
| 小型爬虫 | 靠近玩家后扑咬 | M2 |
| 骷髅兵 | 慢速靠近并近战攻击 | M2 |
| 弓箭骷髅 | 保持距离并远程射击 | M3 |
| 精英盾卫 | 正面减伤，近距离重击 | M3 |
| 遗迹看守者 | Boss，多阶段攻击 | M4 |

### 怪物通用需求

- 血量
- 受击
- 死亡
- 掉落
- 简单寻路或横向追踪
- 攻击冷却

## 8. Boss 设计

### 第一章 Boss：遗迹看守者

遗迹看守者是一具被遗物污染的巨大铠甲。

技能：

1. 横扫：近距离大范围攻击。
2. 跳砸：跳起后砸向玩家位置。
3. 冲撞：横向高速移动。
4. 召唤碎石：第二阶段触发。
5. 狂暴：低血量后攻击频率提高。

击败后掉落核心遗物。

## 9. 掉宝系统

怪物死亡后可能掉落：

- 金币
- 血瓶
- 装备
- 材料
- 遗物
- 技能符文

MVP 阶段优先实现：金币、血瓶、装备。

## 10. 装备系统

MVP 阶段装备部位：

- 武器
- 头盔
- 护甲
- 戒指

装备品质：

| 品质 | 名称 | 说明 |
|---|---|---|
| Common | 普通 | 基础属性 |
| Magic | 精良 | 1-2 条词缀 |
| Rare | 稀有 | 2-4 条词缀 |
| Legendary | 传说 | 有特殊效果 |
| Relic | 遗物 | 改变玩法机制 |

## 11. 词条方向

基础词条：

- 攻击力
- 最大生命值
- 暴击率
- 暴击伤害
- 移动速度
- 攻击速度
- 技能冷却缩减

特殊词条：

- 攻击概率造成流血。
- 闪避后下一击暴击。
- 击杀敌人恢复生命。
- 低生命时获得护盾。
- 跳跃攻击造成范围伤害。

## 12. 遗物系统

遗物是改变玩法的核心系统。

示例遗物：

| 遗物 | 效果 |
|---|---|
| 雷鸣核心 | 普通攻击概率召唤落雷 |
| 血牙项链 | 暴击时恢复生命 |
| 炎魔之眼 | 技能命中附加燃烧 |
| 空洞之翼 | 增加一次跳跃，但降低生命上限 |
| 诅咒金币 | 金币越多伤害越高，但受到伤害也提高 |

## 13. 关卡结构

建议采用房间式横板关卡：

```text
出生房间 → 普通战斗房 → 宝箱/商店/事件房 → 精英房 → Boss 房
```

MVP 阶段先做固定测试关卡，不急着做随机生成。待战斗闭环稳定后再加入房间随机连接。

## 14. 美术风格

建议方向：暗黑像素风或简化手绘 2D。

关键词：

- 暗黑
- 废墟
- 遗物
- 深渊
- 怪物
- 火光
- 诅咒
- 地下城

## 15. MVP 范围

第一版可玩 Demo 包含：

- 一个可移动角色
- 一个测试关卡
- 三种普通怪
- 一个精英怪
- 一个 Boss
- 基础攻击、跳跃、闪避
- 血量系统
- 掉落系统
- 简单装备拾取
- 基础 UI
- 死亡和重开

不在 MVP 范围内：

- 联机
- 完整剧情
- 多角色
- 商店系统
- 大量随机事件
- 云存档
- 排行榜
