# 开发路线图：Relicbound

最后更新：2026-06-25

## 总体目标

先完成一个 5-10 分钟可玩的 2D 横板动作刷宝 Demo，再逐步扩展装备、遗物、关卡随机、Boss 和长期成长。

路线图遵循一个原则：

```text
先闭环，再扩展；先可玩，再丰富；先手感，再内容。
```
